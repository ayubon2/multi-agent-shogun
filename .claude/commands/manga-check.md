## 漫画ファイル命名規則チェッカー

指定ディレクトリ内のファイルが命名規則に従っているかを検証し、違反を報告する。

### 命名規則
```
(カテゴリ) [著者名] タイトル 第XX巻.ext
```
- **カテゴリ**: 一般コミック / 異世界コミック / BLコミック / 少女コミック / 青年コミック 等
- **著者名**: 半角ブラケット `[]` で囲む。複数著者は `×` で区切り
- **タイトル**: 日本語タイトル
- **巻数**: `第XX巻`（2-3桁ゼロ埋め）
- **拡張子**: .zip / .rar / .7z / .cbz / .cbr
- **例**: `(一般コミック) [CLAMP] カードキャプターさくら 第01巻.zip`

### 手順

1. ユーザーが `$ARGUMENTS` でディレクトリパスを渡す（未指定時は確認する）
2. **アクセス制御**: パスが以下のいずれかの配下であることを検証。それ以外は拒否:
   - `/Volumes/Manga01/Manga/`
   - `/Volumes/Manga02/Manga/`
   - `/Volumes/Manga02/一時/`
   - `/Users/nobi/projects/002_manga-system/`
3. 対象ディレクトリ内のファイル一覧を取得（サブディレクトリ1階層まで）
4. 各ファイルを以下のパターンでチェック:
   ```
   ^(\(.+?\)) \[.+?\] .+ 第\d{2,3}巻\.(zip|rar|7z|cbz|cbr)$
   ```
5. 結果を以下の形式で報告:

### 報告形式

```
=== 命名規則チェック結果 ===
対象: /path/to/directory
合計: XX件

✅ 正規形式: XX件
❌ 違反: XX件

--- 違反ファイル ---
1. [欠落:カテゴリ] filename.zip
   → 推奨: (一般コミック) [著者] タイトル 第01巻.zip
2. [欠落:著者] filename.zip
3. [文字化け] ??filename.zip
4. [ローマ字] [author]_title_v01.zip
   → 002_manga-system の manga renamer で変換可能
```

### 違反タイプ
| コード | 意味 | 対処 |
|--------|------|------|
| 欠落:カテゴリ | `(カテゴリ)` がない | カテゴリを付与 |
| 欠落:著者 | `[著者]` がない | DB検索で著者特定 |
| 欠落:巻数 | `第XX巻` がない | ファイル内容から推定 |
| 文字化け | Shift-JIS→UTF-8変換残り | エンコーディング修正 |
| ローマ字 | 英語/ローマ字ファイル名 | `manga renamer` で変換 |
| 拡張子不正 | 対象外拡張子 | 確認して除外 or 変換 |
| 巻数形式 | v01, Vol.01等の非標準形式 | `第XX巻` に正規化 |

### 修正提案
- 002_manga-system の `manga renamer match` を使った自動修正を提案
- 文字化けは Shift-JIS → UTF-8 変換スクリプトを提案
- ローマ字ファイルは DB照合で日本語タイトルを特定

### 注意事項
- **読み取りのみ**: このスキルはチェックのみ。ファイル名の変更は行わない
- **Resilio Sync**: タイムスタンプ変更操作は禁止
- **アクセス制御**: 許可ディレクトリ外のパスは拒否する
