# /go — 統合コマンドメニュー

殿が覚えるコマンドはこれ1つだけ。選択式で全スキルを呼び出す。

## 手順

1. `AskUserQuestion` で以下の選択肢を提示する:

**question**: 「何をしますか？」
**header**: コマンド
**options**:

| label | description |
|-------|-------------|
| システム確認 | 活動モニター + 異常チェック（稼働状態と停滞検知をまとめて） |
| 判断要請 | 殿の判断が必要な事項を選択肢形式で提示 |
| 戦況確認 | dashboard.mdの内容を読みやすく要約 |
| 技術スカウト | 開発改善・漫画サイト・Agent SDK等の有望ツールを探索・評価 |
| 設計書 | サイトの方針・コンセプト・設計判断を閲覧・更新 |

2. 殿の選択に応じて、対応するスキルを `Skill` ツールで実行する:

| 選択 | 実行するスキル |
|------|--------------|
| システム確認 | `activity` を実行した後、続けて `check` を実行する |
| 判断要請 | `decide` |
| 戦況確認 | `status` |
| 技術スカウト | `scout` |
| 設計書 | `design` |
| (Other) | 殿の入力内容に応じて最適なスキルを判断して実行。該当なければ通常会話で対応 |
