# /go — 統合コマンドメニュー

殿が覚えるコマンドはこれ1つだけ。選択式で全スキルを呼び出す。

## 手順

1. `AskUserQuestion` で以下の選択肢を提示する:

**question**: 「何をしますか？」
**header**: コマンド
**options**:

| label | description |
|-------|-------------|
| 活動モニター | エージェント全員の稼働状態を確認（CPU・プロセス・ファイル変更） |
| 異常チェック | 全軍の停滞・異常・ブロッカーを一斉検知（18項目） |
| 判断要請 | 殿の判断が必要な事項を選択肢形式で提示 |
| 戦況確認 | dashboard.mdの内容を読みやすく要約 |

2. 殿の選択に応じて、対応するスキルを `Skill` ツールで実行する:

| 選択 | 実行するスキル |
|------|--------------|
| 活動モニター | `activity` |
| 異常チェック | `check` |
| 判断要請 | `decide` |
| 戦況確認 | `status` |
| (Other) | 殿の入力内容に応じて最適なスキルを判断して実行。該当なければ通常会話で対応 |
