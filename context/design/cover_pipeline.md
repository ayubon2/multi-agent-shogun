# カバー画像パイプライン

決定日: 2026-03-01
ステータス: 実行中（BookWalker 収集開始済み）

## パイプライン優先順

| 優先度 | ソース | 方式 | 品質 | 状態 |
|--------|--------|------|------|------|
| ① | BookWalker | ISBN 検索 → 書影URL | 700x780 高品質 | 実行中 (PID 98917) |
| ② | 紀伊國屋 | ISBN→URL 直接構築（HEAD only） | 中〜高 | 実行中 (PID 96021, 57K件) |
| ③ | 集英社 CloudFront | ISBN→URL 直接構築 | 中（80%が15KB+） | 条件付き承認済み（フラグ管理） |
| ④ | 楽天ブックス API | 公式 API | ~200px（要画質チェック） | 2026-03-02 02:00 自動起動 |
| ⑤ | Amazon PA-API | 公式 API | 最高品質 | 将来（適格販売3件が前提） |

## 除外確定ソース

| ソース | 理由 |
|--------|------|
| AniList | データ汚染（画像の誤マッチ、推しの子→フリーレン事件） |
| Google Books | 低品質 |
| OpenBD | サービス停止（全 URL 404） |
| 出版社サイト（集英社CF除く） | アフィリエイトに繋がらない |
| COCORO BOOKS | API なし、非実用的 |
| JPRO | 個人契約不可 |

## 品質基準（絶対ルール）

- 最低ファイルサイズ: **15KB 以上**（未満は DB 登録しない）
- 最低解像度: **200px 以上**（幅 or 高さ）
- 取得後に HEAD で Content-Length 確認し、基準未満は破棄
- 全ソース共通で適用

## 集英社 CloudFront 特例ルール（2026-03-01 殿判断）

- **条件付き承認**: 短期的な見た目改善のため、集英社のみ出版社書影を例外的に取得
- **フラグ管理必須**: `cover_source` に `shueisha_cf` を記録し、後で BW/楽天書影に差し替え可能にする
- **差し替え方針**: BW 書影が取得できた作品から順次置換し、最終的に集英社 CF 書影は全件削除
- **URL 形式**: `https://dosbg3xlm0x1t.cloudfront.net/images/items/{ISBN}/240/{ISBN}.jpg`
- **対象**: 集英社系 + カバーなし = 約 3,761 件

## メタデータ同時収穫

カバー画像だけでなく、同時に以下も取得する方針:
- あらすじ（synopsis）
- ジャンル・タグ（→ タグ体系に変換）
- レビュー数・評価（ランキング材料）
